📌 Descrição

Este projeto apresenta um dashboard interativo desenvolvido no Power BI para análise de chamados de suporte de TI.
O objetivo é fornecer uma visão clara e rápida sobre o volume de tickets, permitindo a identificação de padrões e acompanhamento do desempenho do suporte.

🎯 Funcionalidades do Dashboard

✅ KPI principal: Total de Tickets
✅ Distribuição de chamados por categoria de problema
✅ Análise de tickets por status (Resolvido vs Pendente)
✅ Visualização simples e intuitiva com design moderno em tema escuro


📊 Métricas Analisadas

Total de chamados registrados
Quantidade de tickets:

Resolvidos
Pendentes


Classificação dos chamados por tipo:

Sistema
Rede
Hardware




🎨 Design
O dashboard segue boas práticas de visualização de dados:

Tema Dark Mode (escuro)
Alto contraste para melhor leitura
Layout clean e minimalista
Hierarquia visual bem definida (KPI → gráficos)


🛠️ Ferramentas Utilizadas

Power BI Desktop
Modelagem de dados simples
Criação de visuais (Card, Pie Chart, Bar Chart)
